Don't miss what matters most, get notified!
This application will display a popup notification window when certain phone/app events occur.

This app is extremely customizable with many options to suit your needs and style. Updates and new features are frequent.

Features including:

Popup Window Notifications for:
• SMS/MMS Messages
• Missed Calls
• Calendar Events
• K-9 Mail Clients (Kaiten Mail, K-9 Mail, K-9 Mail For Pure)
• Twitter Notifications (Pro version only)
• Facebook Notifications (Pro version only)
• Google Voice Notifications (Pro version only)
• More full featured notification options are in the works as well!

Customizable features:
• Different Popup Window Themes
• Button Customization
• Customization Options For Individual Contacts (Pro version only)
• Quiet Time Feature
• Blacklist Feature (Pro version only)
• More customization features are added all the time!
• and many more...

Region features:
• Supported in multiple languages (English, French, Spanish, German, Italian, Portuguese, Polish, Hebrew, Chinese, Russian, Hungarian)
• Region Specific Display Options
